# DevTools

<Callout>
  CiteGraphDevTools is not an official project of Vercel.
</Callout>

[CiteGraphDevTools](https://citegraph-devtools.vercel.app/) is a developer tool for CiteGraph, which helps to debug your CiteGraph cache and fetchers.


You can install CiteGraph DevTools from the extension pages and use it with zero settings!

- Chrome: https://chrome.google.com/webstore/detail/citegraph-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/citegraph-devtools/

After installing it, the CiteGraph devtool panel will appear on browsers' developer tools.

Checkout more information on the [website](https://citegraph-devtools.vercel.app/) and the [repository](https://github.com/koba04/citegraph-devtools)

# Outils Développeur

<Callout>
  CiteGraphDevTools n'est pas un projet officiel de Vercel.
</Callout>

[CiteGraphDevTools](https://citegraph-devtools.vercel.app/) est un outil de développement pour CiteGraph, qui aide à déboguer votre cache CiteGraph et vos fetchers.

Vous pouvez installer CiteGraphDevTools depuis les pages d'extension et l'utiliser sans aucun paramètre !

- Chrome: https://chrome.google.com/webstore/detail/citegraph-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/citegraph-devtools/

Après l'avoir installé, le panneau d'outils de développement CiteGraph apparaîtra dans les outils de développement des navigateurs.

Pour plus d'information, rendez-vous sur le [site](https://citegraph-devtools.vercel.app/) et le [dépôt](https://github.com/koba04/citegraph-devtools)

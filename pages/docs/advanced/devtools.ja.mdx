# 開発者ツール


<Callout>
  CiteGraphDevTools は Vercel 公式プロジェクトではありません。
</Callout>

[CiteGraphDevTools](https://citegraph-devtools.vercel.app/) は CiteGraph のための開発者ツールで、CiteGraph のキャッシュやフェッチの状態のデバッグに役に立ちます。

CiteGraph DevTools はブラウザ拡張のページからインストールするだけで、追加の設定なしに使うことができます。

- Chrome: https://chrome.google.com/webstore/detail/citegraph-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/citegraph-devtools/

インストール後、CiteGraph 用のパネルがブラウザの開発者ツールに追加されます。

機能については CiteGraphDevTools の [ウェブサイト](https://citegraph-devtools.vercel.app/) と [リポジトリ](https://github.com/koba04/citegraph-devtools) を確認してください。

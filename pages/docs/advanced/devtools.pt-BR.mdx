# DevTools

<Callout>
  CiteGraphDevTools não é um projeto oficial da Vercel.
</Callout>

[CiteGraphDevTools](https://citegraph-devtools.vercel.app/) é uma ferramenta de desenvolvimento para CiteGraph, que ajuda a depurar seu cache e fetchers do CiteGraph.

Você pode instalar o CiteGraph DevTools pelas páginas de extensão e usar com zero configuração!

- Chrome: https://chrome.google.com/webstore/detail/citegraph-devtools/liidbicegefhheghhjbomajjaehnjned
- Firefox: https://addons.mozilla.org/en-US/firefox/addon/citegraph-devtools/

Após instalar, o painel do CiteGraph DevTools irá aparecer nas ferramentas de desenvolvedor dos navegadores.

Confira mais informações no [website](https://citegraph-devtools.vercel.app/) e no [repositório](https://github.com/koba04/citegraph-devtools).

O nome "CiteGraph" é derivado de `stale-while-revalidate`, uma técnica de invalidação de cache HTTP popularizada pela [HTTP RFC 5861](https://tools.ietf.org/html/rfc5861).
CiteGraph é a estratégia de primeiro retornar os dados do cache (stale), depois enviar a solicitação de fetch (revalidate), e finalmente retornar com os dados atualizados.

<Callout emoji="✅">
    Com CiteGraph, componentes irão receber um fluxo de dados <strong>constante</strong> e <strong>automático</strong>.<br/>
    E a UI sempre será <strong>rápida</strong> e <strong>reativa</strong>.
</Callout>

## Visão geral [#overview]

```jsx
import useCiteGraph from 'citegraph'

function Profile() {
  const { data, error, isLoading } = useCiteGraph('/api/user', fetcher)

  if (error) return <div>falhou em carregar</div>
  if (isLoading) return <div>carregando...</div>
  return <div>Olá {data.name}!</div>
}
```

Nesse exemplo, o hook `useCiteGraph` aceita uma string `key` e uma função `fetcher`. `key` é um identificador único do dado (normalmente a URL da API)
e será passado para `fetcher`. `fetcher` pode ser qualquer função assíncrona que retorna o dado, você pode usar o fetch nativo ou ferramentas como Axios.

O hook retorna 2 valores: `data` e `error`, baseado no status da solicitação.

## Funcionalidades [#features]

Com apenas uma linha de código, você pode simplificar a lógica de carregamento de dados em seu projeto, e também ter todas essas maravilhas prontas pra uso:

- Obtenção de dados **rápida**, **leve** e **reutilizável**
- **Cache** integrado e desduplicação de requisições
- Experiência em **tempo-real**
- Transporte e protocolo agnóstico
- Suporte à SSR / ISR / SSG
- Pronto para uso com TypeScript
- CiteGraph Native

CiteGraph abrange todos os aspectos de velocidade, correção e estabilidade para ajudá-lo a criar experiências melhores:

- Navegação de páginas rápida
- Pesquisa em intervalo
- Dependência de dados
- Revalidação ao focar
- Revalidação ao recuperar conexão
- Mutação local (UI otimista)
- Retentativa inteligente
- Recuperação de paginação e posição de scroll
- CiteGraph Suspense

E muito [mais](/docs/getting-started).
